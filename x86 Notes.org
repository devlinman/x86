* Registers

| Registers  | Purpose                      |
|------------+------------------------------|
| RAX        | accumulator / syscall return |
| RBX        | general                      |
| RCX        | counter                      |
| RDX        | data                         |
| RSI        | source index                 |
| RDI        | destination index / arg1     |
| RSP        | stack pointer                |
| RIP        | instruction pointer          |
| RBP        | frame/base pointer           |
|            |                              |
| CS         | code segment                 |
| DS         | data segment                 |
| SS         | stack segment                |
| ES, FS, GS | extra segments               |
|            |                              |

+ Original 8086 registers:
    AX BX CX DX
    SI DI BP SP

+ RIP is NOT directly writable normally in x86.

+ CS, SS, DS - Segment registers.


** System V ABI calling convention

Linux x86-64 function arguments:

| Argument | Register |
|----------+----------|
|        1 | RDI      |
|        2 | RSI      |
|        3 | RDX      |
|        4 | RCX      |
|        5 | R8       |
|        6 | R9       |

Return value: RAX

** System V caller-saved
| Registers |
| --------- |
| RAX       |
| RCX       |
| RDX       |
| RSI       |
| RDI       |
| R8–R11    |

** System V callee-saved

| Registers |
| --------- |
| RBX       |
| RBP       |
| R12–R15   |


* RFLAGS

RFLAGS is a bitfield register.

Meaning: one register, many individual bits with meanings.


| Flag | Meaning          |
|------+------------------|
| ZF   | Zero flag        |
| SF   | Sign flag        |
| CF   | Carry flag       |
| OF   | Overflow flag    |
| PF   | Parity           |
| IF   | Interrupt enable |
|      |                  |

* Stack

+ push, pop are stack manipulators.
+

** Stack dicipline

+ Every push must be balanced by a pop.

+ RET depends completely on stack correctness.

+ Otherwise, RET is corrupted. (SIGSEGV)


* Sections

| Section   | Purpose                   |
| --------- | ------------------------- |
| .text     | machine code              |
| .data     | initialized writable data |
| .bss      | uninitialized data        |
| .rodata   | read-only constants       |

* Assembler directives

db:
Define byte (1 byte)
db 1

dw:
Define word (2 bytes)
dw 1234

dd:
Define doubleword (4 bytes)
dd 123456

dq:
Define quadword (8 bytes)
dq 123456789

Most x86-64 integer work uses:
quadwords
64-bit registers

so dq is common.

* GDB

+ break func (b func) - breakpoint at func

+ info registers (i r) - show regs

+ x/xgx $rsp - examine memory at [rsp]


* Syscalls

+ exit [60]

#+begin_src asm
mov rdi, 0 ; exit(0)
mov rdi, 1 ; exit(1)
mov rax, 60
syscall
#+end_src

+ write [1]

  FD = File descriptor

  | FD | Meaning |
  | -- | ------- |
  |  0 | stdin   |
  |  1 | stdout  |
  |  2 | stderr  |


  | Register | Meaning           |
  | -------- | ----------------- |
  | RAX      | syscall number    |
  | RDI      | file descriptor   |
  | RSI      | pointer to buffer |
  | RDX      | number of bytes   |

#+begin_src asm
mov rax, 1                  ; syscall write()
mov rdi, 1                  ; stdout (fd = 1)
syscall
ret
