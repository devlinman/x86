* Registers

| Registers | Purpose                      |
|-----------+------------------------------|
| RAX       | accumulator / syscall return |
| RBX       | general                      |
| RCX       | counter                      |
| RDX       | data                         |
| RSI       | source index                 |
| RDI       | destination index / arg1     |
| RSP       | stack pointer                |
| RIP       | instruction pointer          |
