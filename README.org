* Registers

| Register   | Purpose                          |
|------------+----------------------------------|
| RAX        | accumulator / return value       |
| RBX        | general-purpose                  |
| RCX        | counter                          |
| RDX        | data / syscall arg3              |
| RSI        | source index / syscall arg2      |
| RDI        | destination index / syscall arg1 |
| RSP        | stack pointer                    |
| RBP        | frame/base pointer               |
| RIP        | instruction pointer              |
| R8-R15     | additional general registers     |
|            |                                  |
| CS         | code segment                     |
| DS         | data segment                     |
| SS         | stack segment                    |
| ES, FS, GS | extra segment registers          |

+ Original 8086 registers:
  - AX BX CX DX
  - SI DI BP SP

+ RIP is not directly writable using MOV.

+ RIP changes via: =jmp=, =call=, =ret=, interrupts/exceptions

+ In x86_64 Linux:
  - segmentation is mostly unused
  - FS/GS are still important
  - FS is commonly used for thread-local storage


*** Subregisters:

| 64-bit | 32-bit | 16-bit | 8-bit |
|--------+--------+--------+-------|
| RAX    | EAX    | AX     | AL    |
| RBX    | EBX    | BX     | BL    |
| RCX    | ECX    | CX     | CL    |
| RDX    | EDX    | DX     | DL    |

+ Writing to a 32-bit register zero-extends the upper 32 bits.

  #+begin_src asm
  mov eax, 1
  ; RAX becomes 1
  ; upper 32 bits become 0
  #+end_src

+ AL only writes to the lower portion while upper portion remains unchanged.

** System V ABI calling convention

+ Linux x86_64 function arguments:

| Argument | Register |
|----------+----------|
|        1 | RDI      |
|        2 | RSI      |
|        3 | RDX      |
|        4 | RCX      |
|        5 | R8       |
|        6 | R9       |

+ Return values: RAX

+ Additional arguments: passed on the stack

+ Stack alignment: stack must be 16-byte aligned before CALL

* RFLAGS

RFLAGS is a bitfield register.

One register containing many individual flag bits.

| Flag | Meaning                    |
|------+----------------------------|
| ZF   | Zero flag                  |
| SF   | Sign flag                  |
| CF   | Carry flag                 |
| OF   | Overflow flag              |
| PF   | Parity flag                |
| IF   | Interrupt enable flag      |

#+begin_src asm
cmp rax, rbx
je equal
#+end_src

+ =cmp= sets flags, with no modifications to registers

+ =je= checks ZF

| Operation | Example                |
|-----------+------------------------|
| mov       | mov rax, rbx           |
| add       | add rax, rbx           |
| sub       | sub rax, rbx           |
| imul      | imul rax, rbx          |
| xor       | xor rax, rax           |

| call      | call func              |

| syscall   | syscall                |

+ =xor rax, rax= is a commonly used idiom to zero a register.

+ =lea= computes addresses.

+ =test rax, rax= checks whether RAX is zero without modifying it.


** Signed vs Unsigned

Signed jumps: =jl=, =jg=

Unsigned jumps: =jb=, =ja=


** Endianness

+ x86 is little-endian.

#+begin_src text
0x12345678
#+end_src

+ Stored in memory as:

#+begin_src text

#+end_src

* Memory Addressing

+ Address of msg:

#+begin_src asm
mov rax, msg
#+end_src

+ Value stored at msg:

#+begin_src asm
mov rax, [msg]
#+end_src

+ Read memory at address stored in RBX:

#+begin_src asm
mov rax, [rbx]
#+end_src

+ Write RAX into memory pointed to by RBX:

#+begin_src asm
mov [rbx], rax
#+end_src


+ Addressing forms:

#+begin_src asm
mov rax, [rbx]
mov rax, [rbx+8]
mov rax, [rbx+rcx*4]
#+end_src

* Stack

+ Stack grows downward:
  - pushing decreases RSP
  - popping increases RSP

+ Stack-manipulation instructions: =push=, =pop=, =call=, =ret=.

Push to stack:

#+begin_src asm
push rax

;; Alternate Method
sub rsp, 8
mov [rsp], rax
#+end_src

Pop from stack:

#+begin_src asm
pop rax

;; Alternate Method
mov rax, [rsp]
add rsp, 8
#+end_src

+ CALL internally does:
  + push return address
  + jump to target

+ RET internally does:
  + pop return address into RIP


** Stack discipline

+ Every PUSH should eventually be balanced by POP.

+ Incorrect stack management can corrupt:
  + return addresses
  + local variables
  + saved registers

+ Result: crashes, undefined behavior, SIGSEGV

+ Function prologue:

#+begin_src asm
push rbp
mov rbp, rsp
sub rsp, 32
#+end_src

+ Function epilogue:

#+begin_src asm
mov rsp, rbp
pop rbp
ret
#+end_src

** Assembler directives

+ db: define byte (1 byte)

#+begin_src asm
db 1
#+end_src

+ dw: define word (2 bytes)

#+begin_src asm
dw 1234
#+end_src

+ dd: define doubleword (4 bytes)

#+begin_src asm
dd 123456
#+end_src

+ dq: define quadword (8 bytes)

#+begin_src asm
dq 123456789
#+end_src

+ Common in x86_64:
  + 64-bit registers
  + 64-bit pointers
  + quadwords (=dq=)

+ Reserve-space directives:

| Directive | Meaning            |
|-----------+--------------------|
| resb      | reserve bytes      |
| resw      | reserve words      |
| resd      | reserve dwords     |
| resq      | reserve qwords     |

+ Usually used in: =.bss=

* Syscalls

Linux x86_64 syscall convention:

| Register | Meaning        |
|----------+----------------|
| RAX      | syscall number |
| RDI      | arg1           |
| RSI      | arg2           |
| RDX      | arg3           |
| R10      | arg4           |
| R8       | arg5           |
| R9       | arg6           |

Return value: RAX


** exit [60]

#+begin_src asm
mov rax, 60
mov rdi, 0                      ; exit code = rdi = 0
syscall
#+end_src


** write [1]

+ File descriptors:

| FD | Meaning |
|----+---------|
|  0 | stdin   |
|  1 | stdout  |
|  2 | stderr  |

+ Arguments:

| Register | Meaning           |
|----------+-------------------|
| RDI      | file descriptor   |
| RSI      | pointer to buffer |
| RDX      | number of bytes   |

#+begin_src asm
mov rax, 1
mov rdi, 1                      ; fd = stdout
mov rsi, msg                    ; pointer to bytes
mov rdx, msg_len                ; length of message bytes
syscall
#+end_src


** read [0]

+ Returns number of bytes read in RAX

#+begin_src asm
mov rax, 0
mov rdi, 0                      ; fd = rdi = stdin
mov rsi, buffer                 ; address of buffer
mov rdx, 64                     ; Max. bytes to read
syscall
#+end_src
